# niehewang.github.io

Static GitHub Pages site for Hewang Nie.

## Pages

- `index.html` - bilingual academic homepage.
- `ccf-2026.html` - CCF 2026 journal and conference lookup tool.
- `research-workbench.html` - local-only personal research workbench.
- `404.html` - GitHub Pages fallback page.

## Assets

- `assets/site.css` - shared homepage visual system.
- `assets/site.js` - homepage interactions: theme, language switch, publication search, scroll tools.
- `assets/favicon.svg` - site icon.
- `photo.jpg` - homepage portrait.
